## Stack

Single-file static HTML + CSS + vanilla JS. No build step. Hosted on Vercel.

Highlights:
- Editorial serif + clean sans typography (Fraunces + Inter, per font-library pairing #2)
- Warm graphite + saffron palette - distinct from prior prospect builds (no recycled theme)
- Hero focused on the mobile promise ("The interior you forgot your car had")
- Trust stack pulls real Google review data: 4.9 stars, 173 reviews, 5+ years
- 4-card service grid using Issiah's actual Wix CDN imagery
- Process timeline (4 steps: text a photo - pick a slot - we pull up - you walk out)
- Real featured testimonials lifted verbatim from his Google profile
- Service-area block with Sacramento metro + Roseville/Rocklin + Yolo/Solano SEO list
- 7-item FAQ addressing real buyer objections (hose? garage? smoke smell? who's home?)
- LocalBusiness + AggregateRating JSON-LD for local SEO
- Click-to-call + SMS fast path
- `prefers-reduced-motion` respected, WCAG AA contrast, keyboard nav
